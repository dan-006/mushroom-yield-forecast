\# Capstone Reflection



\## Top 3 Skills Learned



\### 1. End-to-End Machine Learning Workflow



I learned how to build a complete machine learning pipeline from raw data ingestion through cleaning, feature engineering, model training, evaluation, deployment, and monitoring.



\### 2. Model Evaluation and Validation



I gained experience using temporal train-test splits, cross-validation, MAE, RMSE, and R² metrics to compare models and avoid data leakage.



\### 3. Deployment and Monitoring



I learned how to deploy a machine learning model using Streamlit Community Cloud and implement logging, testing, and monitoring practices for production-ready applications.



\## Areas for Growth



\### 1. Advanced Feature Engineering



Future work could include creating more sophisticated environmental features and incorporating additional sensor data sources.



\### 2. MLOps and Automation



I would like to learn more about automated retraining pipelines, model versioning, and cloud-based monitoring systems.



\## Final Reflection



This project provided hands-on experience with the complete data science lifecycle. Building a forecasting system from raw sensor data through deployment strengthened both my technical and problem-solving skills. The project also highlighted the importance of validation, reproducibility, monitoring, and clear communication when delivering machine learning solutions.
