# Capstone Reflection



## Top 3 Skills Learned



### 1. End-to-End Machine Learning Workflow



I learned how to build a complete machine learning pipeline, starting from raw data ingestion and preprocessing through feature engineering, model training, evaluation, deployment, and monitoring.



### 2. Model Evaluation and Validation



I gained experience applying temporal train-test splits, cross-validation, and evaluation metrics such as MAE, RMSE, and R² to assess model performance while avoiding data leakage.



### 3. Deployment and Monitoring



I learned how to deploy a machine learning application using Streamlit Community Cloud and implement logging, testing, and monitoring practices to support a production-ready workflow.



## Areas for Growth



### 1. Advanced Feature Engineering



Future work could include developing more sophisticated environmental features and incorporating additional sensor data sources to improve predictive performance.



### 2. MLOps and Automation



I would like to deepen my knowledge of automated retraining pipelines, model versioning, CI/CD practices, and cloud-based monitoring systems.



## Final Reflection



This project provided valuable hands-on experience across the complete data science lifecycle. Building a mushroom yield forecasting system from raw sensor data through deployment strengthened both my technical and problem-solving skills. The project also emphasized the importance of validation, reproducibility, monitoring, and clear communication when delivering machine learning solutions in real-world environments.
